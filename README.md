Laboratorio 4

Descripción general:
Este proyecto contiene una serie de scripts desarrollados en PowerShell
que permiten obtener información del sistema, gestionar procesos y automatizar tareas.

Principales funciones:
- Obtener la información del sistema.
- Listar procesos en ejecución.
- Eliminar un proceso por ID o nombre.
- Crear carpetas donde su nombre sea la fecha actual.
- Un menú interactivo para acceder a distintas funciones.

Scripts:
- reto1.ps1: Muestra la información del sistema.
- reto2.ps1: Lista los procsos y permite eliminar uno.
- reto3.ps1: Crea una carpeta y que el nombre sea la fecha actual
- menu.ps1: Un menú con 3 diferentes opciones.

Instrucciones de ejecución:
1- Abrir Visual Studio Code.
2- Abrir la carpeta del proyecto.
3- Abrir el archivo que deseas ejecutar.
4- Abrir la terminal integrada.
5- Ejecutar el script con el comando:
    
    .\<Nombre del archivo>.ps1
